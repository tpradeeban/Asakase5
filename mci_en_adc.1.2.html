MCI Details

The MCIs are detailed below. 

    UTL_GetAdcHandle --- Obtaining the Adc Handle

    UTL_SetAdcPgEnable --- Specifying a Permission of ADC Start Function for the Adc Handle

    UTL_SetAdcStartCycle --- Sets an ALPG Signal to Start ADC in an Adc Handle

    UTL_SetAdcInput --- Specifying a Unit to Measure for Adc Handle

    UTL_SetAdcPinList --- Setting Pinlist Name to Adc Handle

    UTL_SetAdcMeasMode --- Setting Measurement Mode for ADC Start to Adc Handle

    UTL_SetAdcHistoryMode --- Setting a History Mode in an Adc Handle

    UTL_SendAdc --- Sending the Adc Handle Settings to the Test System

    UTL_WaitAdc --- Waiting for Completion of ADC Operation

    UTL_ResetAdcHistoryCount --- Resetting History Entry Count in DC Test

    UTL_GetAdcDutGroupMaxNumber --- Obtaining Largest DUT Group Number for DC Test that Uses ADC Start Function

    UTL_WriteAdcDutGroupNumber --- Setting DUT Group Number for DC Test that Uses ADC Start Function

    UTL_ReadAdcDutGroupNumber --- Reading DUT Group Number for DC Test that Uses ADC Start Function

    UTL_GetAdcDutGroupDutCursor --- Obtaining DUT Numbers that Belong to DUT Group as DUT Cursor
