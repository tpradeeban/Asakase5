UTL_GetUbmConfigPatternInvEnable --- Obtaining the PatternInvEnable that Is Set to the Handle

SYNTAX

int UTL_GetUbmConfigPatternInvEnable( UbmConfigHandle h, CheckButton *onoff );

ARGUMENT(S)

h : UbmConfigHandle

onoff : Pointer of the variable in which the PatternInvEnable is stored

RETURN VALUE

  • 1 : Read correctly.

  • 0 : No data exists in the handle.

  • -1 : Could not be read.(An error occurred.)

DESCRIPTION

This MCI is used to obtain the PatternInvEnable that is set to the handle.

ERROR CHECK(S)

  • If an incorrect handle is specified, a runtime error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained (No change).
