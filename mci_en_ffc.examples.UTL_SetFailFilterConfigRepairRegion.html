Example of using UTL_SetFailFilterConfigRepairRegion()

If each main(core) is 1024 columns wide and each spare is 81 columns wide, the page size is 4 x (1024 + 81) = 4420.


Example 1

For the above example dividing into 2 repair regions, set up as follows: 



    UTL_SetFailFilterConfigRepairRegion(ffCH, 2, (4420 / 2),  0);
                                              ~~  ~~~~~~~~~   ~~
                                              *a     *b       *c

        *a: Number of main/spare repair regions
        *b: Number of columns per main repair region which includes spare repair region
        *c: No spare repair region


Example 2

For the above example dividing into 2 repair regions, set up as follows: 



    UTL_SetFailFilterConfigRepairRegion(ffCH, 2, (1024 * 2), (81 * 2));
                                              ~~  ~~~~~~~~~   ~~~~~~
                                              *a     *b       *c

        *a: Number of main/spare repair regions
        *b: Number of columns per main repair region
        *c: Number of columns per spare repair region
