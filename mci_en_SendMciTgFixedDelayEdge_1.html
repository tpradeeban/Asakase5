Objective

The MCIs described here are used to specify the Fixed Delay Edge function.

Fixed Delay Edge provides a clock that is output instead of CCLK in the RZO, RZZ, RZX, and DNRZ waveform modes (including inversion modes).

The clock is output according to the specified delay value relative to the reference clock (BCLK).

If Fixed Delay Edge is set, CCLK is masked and not output.

The following shows the Fixed Delay Edge diagram. (Fixed Delay Edge is set for TS1 and TS3 of pins 33 to 40.)


Fixed Delay Edge Diagram

CAUTION

  • These MCIs can be used only for the following test system:

    • T5833

  • Do not set clock output prohibition in the cycle where Fixed Delay Edge is specified.

    If it is set, the following restrictions apply:

    • If setting clock output prohibition to the reference clock (BCLK), it must also be set to CCLK in the identical cycle.

      If clock output prohibition is not set to CCLK, Fixed Delay Edge is generated at an unintended timing.

    • If setting Fixed Delay Edge output prohibition, clock output prohibition must be set to CCLK in the identical cycle.
