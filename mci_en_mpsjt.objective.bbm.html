BBM Test Using Multi-plane Status Judgment Table

The multi-plane status read function stores bad block information in BBM based on status information (8 bits) output from devices.

The following BBM address bits are the plane addresses according to the number of planes:

Number of Planes and BBM Address Bits
Number of planesBBM address bits
1Without bit (Plane 0)
2Lower one bit (Plane 0 to Plane 1)
4Lower two bits (Plane 0 to Plane 3)
8Lower three bits (Plane 0 to Plane 7)

The multi-plane status judgment table is a lookup table that is used to judge bad block information for each plane based on status information output from devices. Judgment data of each plane is stored in BBM by writing it to the table beforehand.

Image of BBM Test Using Multi-plane Status Judgment Table

  1. Specify pin numbers of DUTs from which status information is output.

  2. Specify a setting to capture the judgment result based on status information in BBM.

  3. Specify an MUT signal, which is the same as the status read signal used for the BBM status read cycle, to the pattern program.

  4. Provide the judgment data based on status information in the multi-plane status judgment table.

  5. Status information is used for the addresses in the multi-plane status judgment table.

  6. According to status information, the judgment data items for the number of planes in the relevant addresses of the multi-plane status judgment table are stored in BBM.

Setting example of relevant MCIs 



   // 1.
   UTL_SetFailSelPinTarget( fspH, UT_FAILSELPIN_TARGET_STATUS_FAIL );   // Defines status information.
   UTL_AddFailSelPinAssign( fspH, 0, 1 );                               // DQ0 (P1)
   UTL_AddFailSelPinAssign( fspH, 1, 2 );                               // DQ1 (P2)
   UTL_AddFailSelPinAssign( fspH, 2, 3 );                               // DQ2 (P3)
   UTL_AddFailSelPinAssign( fspH, 3, 4 );                               // DQ3 (P4)
   UTL_AddFailSelPinAssign( fspH, 4, 5 );                               // DQ4 (P5)
   UTL_AddFailSelPinAssign( fspH, 5, 6 );                               // DQ5 (P6)
   UTL_AddFailSelPinAssign( fspH, 6, 7 );                               // DQ6 (P7)
   UTL_AddFailSelPinAssign( fspH, 7, 8 );                               // DQ7 (P8)
       :

   // 2. 6.
   UTL_AddBbmConfigCaptureDataByTargetBank ( bbmH, 0, 1, 0, UT_BBM_STATUSFAIL );    // Specifies a setting to capture the judgment result based on status information in BBM.
   UTL_SetBbmConfigStatusSignal            ( bbmH, UT_SIG_C(5) );                   // Status read signal used for the status read cycle.
   UTL_SetBbmConfigPlaneCount              ( bbmH, 2 );                             // Number of planes: 2
   UTL_SetBbmConfigAddrAssign              ( bbmH, 0,  UT_SIG_X(0) );               // Address signal assigned to BBM  <-- (*1)
   UTL_SetBbmConfigAddrAssign              ( bbmH, 1,  UT_SIG_X(1) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 2,  UT_SIG_X(2) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 3,  UT_SIG_X(3) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 4,  UT_SIG_X(4) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 5,  UT_SIG_X(5) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 6,  UT_SIG_X(6) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 7,  UT_SIG_X(7) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 8,  UT_SIG_X(8) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 9,  UT_SIG_X(9) );               // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 10, UT_SIG_X(10) );              // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 11, UT_SIG_X(11) );              // Address signal assigned to BBM
   UTL_SetBbmConfigAddrAssign              ( bbmH, 12, UT_SIG_X(12) );              // Address signal assigned to BBM
       :
            (*1) Because the number of planes is 2, the lower one bit of the address assigned to BBM is the plane address.
