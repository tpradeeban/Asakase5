Saving UBM Data to a CSV File

UBM data can be saved to a CSV file.

( Operation procedure)

  1. When the [Save Text...] sub menu on the [File] menu in the main window is selected, the file selection dialog box is displayed.


    UBM Tool: File Selection Dialog Box

  2. Specify the file name in the Selection Text Box for saving.

  3. To save to the specified file, click the [OK] button. To cancel, click the [Cancel] button.

If you specify a file name that is the same as the name of an existing file, a warning message appears to prompt you to confirm that you are replacing the existing file with a new one.

To overwrite the previous file, click the [OK] button. Otherwise, click the [Cancel] button.
