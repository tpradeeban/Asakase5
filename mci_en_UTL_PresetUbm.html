UTL_PresetUbm --- Presetting the Universal Buffer Memory by Using the Value in the Argument

SYNTAX

void UTL_PresetUbm( UbmAccessHandle h, USlider n );

ARGUMENT(S)

h : UbmAccessHandle

n : Value that is used for the preset

Setting range for Preset Value.
Tester nameRange
T58330 or 1
T58310 or 1
T58300
T58220 or 1
T57810 or 1
T57610 or 1
T55880
T5503HS20 or 1
T5503HS0 or 1
T53850 or 1
T53830 or 1
W53200 or 1

RETURN VALUE

None

DESCRIPTION

This MCI is used to preset the whole universal buffer memory area by using the value in the n argument.

ERROR CHECK(S)

  • If the specified handle is incorrect, a runtime error occurs.

  • If the specified preset value is incorrect, a runtime error occurs.

  • If the ALPG is not stopped (operating or paused), a runtime error occurs. For the T5503HS2, a runtime error occurs only if the ALPG is operating.

REQUIREMENT(S)

  • UbmAccessHandle must be obtained.

RESULT(S)

  • UbmAccessHandle has been obtained.(No change)

  • The universal buffer memory has been preset by using the n value.
