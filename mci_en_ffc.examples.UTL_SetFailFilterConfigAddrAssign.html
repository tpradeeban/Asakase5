Example of using UTL_SetFailFilterConfigAddrAssign()

Below is an example of 39 address assignments. The maximum number of non-column address assignments is 32 (24 in this example), and this address assignment information is passed to the fail filter when either of the following is satisfied:

  • When it is returned a TRA address in UBM mode.

  • When pulls it from hardware in bypass mode.

Once this has been set, the address can be computed and used for data being sent to the Fail DB. 



    /* 15 address signals for column */
    UTL_SetFailFilterConfigAddrAssign(ffh,  0, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(0));
    UTL_SetFailFilterConfigAddrAssign(ffh,  1, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(1));
    UTL_SetFailFilterConfigAddrAssign(ffh,  2, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(2));
    UTL_SetFailFilterConfigAddrAssign(ffh,  3, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(3));
    UTL_SetFailFilterConfigAddrAssign(ffh,  4, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(4));
    UTL_SetFailFilterConfigAddrAssign(ffh,  5, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(5));
    UTL_SetFailFilterConfigAddrAssign(ffh,  6, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(6));
    UTL_SetFailFilterConfigAddrAssign(ffh,  7, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(7));
    UTL_SetFailFilterConfigAddrAssign(ffh,  8, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(8));
    UTL_SetFailFilterConfigAddrAssign(ffh,  9, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(9));
    UTL_SetFailFilterConfigAddrAssign(ffh, 10, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(10));
    UTL_SetFailFilterConfigAddrAssign(ffh, 11, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(11));
    UTL_SetFailFilterConfigAddrAssign(ffh, 12, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(12));
    UTL_SetFailFilterConfigAddrAssign(ffh, 13, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(13));
    UTL_SetFailFilterConfigAddrAssign(ffh, 14, UT_FF_ADDR_ASSIGN_COLUMN, UT_SIG_Y(14));

    /* 8 address signals for page */
    UTL_SetFailFilterConfigAddrAssign(ffh, 15, UT_FF_ADDR_ASSIGN_PAGE,   UT_SIG_X(0));
    UTL_SetFailFilterConfigAddrAssign(ffh, 16, UT_FF_ADDR_ASSIGN_PAGE,   UT_SIG_X(1));
    UTL_SetFailFilterConfigAddrAssign(ffh, 17, UT_FF_ADDR_ASSIGN_PAGE,   UT_SIG_X(2));
    UTL_SetFailFilterConfigAddrAssign(ffh, 18, UT_FF_ADDR_ASSIGN_PAGE,   UT_SIG_X(3));
    UTL_SetFailFilterConfigAddrAssign(ffh, 19, UT_FF_ADDR_ASSIGN_PAGE,   UT_SIG_X(4));
    UTL_SetFailFilterConfigAddrAssign(ffh, 20, UT_FF_ADDR_ASSIGN_PAGE,   UT_SIG_X(5));
    UTL_SetFailFilterConfigAddrAssign(ffh, 21, UT_FF_ADDR_ASSIGN_PAGE,   UT_SIG_X(6));
    UTL_SetFailFilterConfigAddrAssign(ffh, 22, UT_FF_ADDR_ASSIGN_PAGE,   UT_SIG_X(7));

    /* 2 address signals for plane */
    UTL_SetFailFilterConfigAddrAssign(ffh, 23, UT_FF_ADDR_ASSIGN_PLANE,  UT_SIG_X(8));
    UTL_SetFailFilterConfigAddrAssign(ffh, 24, UT_FF_ADDR_ASSIGN_PLANE,  UT_SIG_X(9));

    /* 11 address signals for block */
    UTL_SetFailFilterConfigAddrAssign(ffh, 25, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(10));
    UTL_SetFailFilterConfigAddrAssign(ffh, 26, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(11));
    UTL_SetFailFilterConfigAddrAssign(ffh, 27, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(12));
    UTL_SetFailFilterConfigAddrAssign(ffh, 28, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(13));
    UTL_SetFailFilterConfigAddrAssign(ffh, 29, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(14));
    UTL_SetFailFilterConfigAddrAssign(ffh, 30, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(15));
    UTL_SetFailFilterConfigAddrAssign(ffh, 31, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(16));
    UTL_SetFailFilterConfigAddrAssign(ffh, 32, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(17));
    UTL_SetFailFilterConfigAddrAssign(ffh, 33, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(18));
    UTL_SetFailFilterConfigAddrAssign(ffh, 34, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(19));
    UTL_SetFailFilterConfigAddrAssign(ffh, 35, UT_FF_ADDR_ASSIGN_BLOCK,  UT_SIG_X(20));

    /* 3 address signals for chip */
    UTL_SetFailFilterConfigAddrAssign(ffh, 36, UT_FF_ADDR_ASSIGN_CHIP,   UT_SIG_X(21));
    UTL_SetFailFilterConfigAddrAssign(ffh, 37, UT_FF_ADDR_ASSIGN_CHIP,   UT_SIG_X(22));
    UTL_SetFailFilterConfigAddrAssign(ffh, 38, UT_FF_ADDR_ASSIGN_CHIP,   UT_SIG_X(23));

UTL_SetFailFilterConfigAddrAssign() allows overlapping block and plane address assignments.

With the example above, the plane would be assigned one bit of the address, and then the block address would be assigned to higher bits of the address. This would result in each block containing two pages.

When the assignment is changed as follows, the plane bit can now overlap the lower block bit. 



    UTL_SetFailFilterConfigAddrAssign(ffh, 17, UT_FF_ADDR_ASSIGN_PAGE,  UT_SIG_X(2));
    UTL_SetFailFilterConfigAddrAssign(ffh, 18, UT_FF_ADDR_ASSIGN_PAGE,  UT_SIG_X(3));
    UTL_SetFailFilterConfigAddrAssign(ffh, 19, UT_FF_ADDR_ASSIGN_PAGE,  UT_SIG_X(4));
    UTL_SetFailFilterConfigAddrAssign(ffh, 20, UT_FF_ADDR_ASSIGN_PAGE,  UT_SIG_X(5));
    UTL_SetFailFilterConfigAddrAssign(ffh, 21, UT_FF_ADDR_ASSIGN_PAGE,  UT_SIG_X(6));
    UTL_SetFailFilterConfigAddrAssign(ffh, 22, UT_FF_ADDR_ASSIGN_PAGE,  UT_SIG_X(7));

    UTL_SetFailFilterConfigAddrAssign(ffh, 23, UT_FF_ADDR_ASSIGN_PLANE, UT_SIG_X(8)); <--------
    UTL_SetFailFilterConfigAddrAssign(ffh, 24, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(8)); <--------

    UTL_SetFailFilterConfigAddrAssign(ffh, 25, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(9));
    UTL_SetFailFilterConfigAddrAssign(ffh, 26, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(10));
    UTL_SetFailFilterConfigAddrAssign(ffh, 27, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(11));
    UTL_SetFailFilterConfigAddrAssign(ffh, 28, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(12));
    UTL_SetFailFilterConfigAddrAssign(ffh, 29, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(13));
    UTL_SetFailFilterConfigAddrAssign(ffh, 30, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(14));
    UTL_SetFailFilterConfigAddrAssign(ffh, 31, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(15));
    UTL_SetFailFilterConfigAddrAssign(ffh, 32, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(16));
    UTL_SetFailFilterConfigAddrAssign(ffh, 33, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(17));
    UTL_SetFailFilterConfigAddrAssign(ffh, 34, UT_FF_ADDR_ASSIGN_BLOCK, UT_SIG_X(18));

    UTL_SetFailFilterConfigAddrAssign(ffh, 35, UT_FF_ADDR_ASSIGN_CHIP,  UT_SIG_X(19));
    UTL_SetFailFilterConfigAddrAssign(ffh, 36, UT_FF_ADDR_ASSIGN_CHIP,  UT_SIG_X(20));
    UTL_SetFailFilterConfigAddrAssign(ffh, 37, UT_FF_ADDR_ASSIGN_CHIP,  UT_SIG_X(21));
