Setting a Radix

To set a radix, select [Hexadecimal] or [Decimal] from the [Radix] sub menu in the [View] menu.


Wave Tracer: [Radix] Menu

[Hexadecimal] is set by default.

The changed displayed mode is valid for the trigger conditions and the following items in the cycle information display area:

  • PC value

  • Address(X,Y,Z), SUB address(X,Y,Z)

  • DBM address

  • UBM address
