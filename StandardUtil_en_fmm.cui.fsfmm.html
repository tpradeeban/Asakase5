fsfmm

  1. Outline

    The fsfmm command is used to display the operating condition of each memory unit such as the FCM and CFM (FM Condition).

    FM conditions can also be saved in a file.

  2. Format 

    


   % fsfmm [ --session <session_id> ] \
           [ --station <station_number> ] \
           [ --site <site_number> ] \
           [ --unit <unit_name> ] \
           [ --output-file <file_name> ] \
           [ -h ]

  3. Basic functions

    This command is used to display FM conditions.

    The following shows an example of use: 

    


   % fsfmm

    • --session <session_id>

      Specify the session ID that uses the command.

      For example, when executing this command with the session user_1, enter the following command: 

      


  % fsfmm --session user_1

      About the case of omitting this option, refer to "Specifying the Session ID".

    • --station <station_number>

      Specify a target station number.

      If this option is omitted, the command is executed assuming that the smallest usable station number has been specified.

      To execute the command targeting Station 1, enter the command as follows: 

      


  % fsfmm --station 1

    • --site <site_number>

      Specify the target site number.

      If this option is omitted, the command is processed assuming that the smallest usable site number has been specified.

      To execute for Site 6, enter the following command: 

      


  % fsfmm --site 6

    • --unit <unit_name>

      Specify the target unit.

      The following unit names can be specified in <unit_name>.

      fsfmm: --unit option
      Unit nameDescription
      fcmDisplays the FCM condition.
      cfmDisplays the CFM condition.
      pmDisplays the PM condition.
      dfmDisplays the DFM condition.
      ubmDisplays the UBM condition.
      ffDisplays the FF condition.
      dbmDisplays the DBM condition.

      If this option is omitted, the command is processed assuming that all units that can be specified have been specified.

      To execute for FCM, enter the following command: 

      


  % fsfmm --unit FCM

    • --output-file <file_name>

      If it is desired to save the FM conditions acquired, specify a file name to save the data.

      To check the content of the saved file, use the fsviewfile command.

      For more information on the fsviewfile command, refer to "FS Command" in "FutureSuite Basic User Interface Operation Manual".

    • -h

      By executing the following command from the shell, the instructions on how to use this command are displayed: 

      


  % fsfmm -h
  Usage: fsfmm [ --session <session_id> ]
               [ --station <station_number> ]
               [ --site <site_number> ]
               [ --unit <unit_name> ]
               [ --output-file <file_name> ]
               [ -h ]
  %
