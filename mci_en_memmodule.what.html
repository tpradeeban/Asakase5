Memory Module

In this manual, the memory module generically refers to the following memories:

  • Fail capture memory (FCM)

  • Pattern memory (PM)

  • Data buffer memory (DBM)

  • Fail buffer memory (FBM)

    FBM is memory used for repair analysis.

The purpose of the memory module (whether to use the memory module as the FCM, PM, etc.) is referred to as the "usage-mode".

Depending on test systems, one memory module contains two or more usage-modes, and it is used by switching those modes. When using such a module, usage-modes must be set before using the memory module. For more information on usage-modes settings, refer to "MCIs that Set Memory Module Usage-Modes".
