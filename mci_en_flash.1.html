Objective

These MCIs are used to set up the flash memory test functions and read the result of flash memory test. During multiple devices in parallel test, excessive writing operations for each DUT need to be prevented.

Therefore, logical comparison and driver output for each DUT have to be disabled in real time.

Logical comparison and driver output can be disabled for the following DUTs.

  • DUTs which all pins are matched

  • DUTs in which the flash counters match the limit registers

  • DUTs which have been masked

Pins with disabled driver output are specified as UTL_SetPinPatternInh() or UTL_SetPinPatternInhPerPin(). Therefore, logical comparison is also prohibited with regard to pins specified as UTL_SetPinPatternInhPerPin().
