Example of using UTL_SetFailFilterConfigPinList()

  • Example 1: Capture 8 bits. 



    UTL_SetFailFilterConfigDutBitSize(ffCH, 8);
    UTL_SetFailFilterConfigBitSize   (ffCH, 8);
    UTL_SetFailFilterConfigPinList   (ffCH, 0, "DQ_0_7", UT_ON, 1); /* ByteGroup=0, deviceChannel=1 */
      :
    UTL_SetFailListAccessDeviceChannel(flAH, 1);  /* deviceChannel=1 */

  • Example 2: Capture 8 bits plus up to 8 non-DQ pins.

    The non-DQ pin list must also be 8 bits. To capture less than 8 bits, just assign the extra pins to drive-only pins and there will be no fail data collected. 



    UTL_SetFailFilterConfigDutBitSize(ffCH, 16);
    UTL_SetFailFilterConfigBitSize   (ffCH, 16);
    UTL_SetFailFilterConfigPinList   (ffCH,  0, "DQ_0_7", UT_ON,  1);
    UTL_SetFailFilterConfigPinList   (ffCH,  1, "non_DQ", UT_OFF, 1);
      :
    UTL_SetFailListAccessDeviceChannel(flAH, 1);  /* deviceChannel=1 */

  • Example 3a: Capture 16 bits. 



    UTL_SetFailFilterConfigDutBitSize(ffCH, 16);
    UTL_SetFailFilterConfigBitSize   (ffCH, 16);
    UTL_SetFailFilterConfigPinList   (ffCH,  0, "DQ_0_15", UT_ON, 1);
      :
    UTL_SetFailListAccessDeviceChannel(flAH, 1);  /* deviceChannel=1 */

  • Example 3b: Capture 16 bits. 



    UTL_SetFailFilterConfigDutBitSize(ffCH, 16);
    UTL_SetFailFilterConfigBitSize   (ffCH, 16);
    UTL_SetFailFilterConfigPinList   (ffCH,  0, "DQ_0_7",  UT_ON, 1);
    UTL_SetFailFilterConfigPinList   (ffCH,  1, "DQ_8_15", UT_ON, 1);
      :
    UTL_SetFailListAccessDeviceChannel(flAH, 1);  /* deviceChannel=1 */

  • Example 4: Capture 8 bits x 4 channels. 



    UTL_SetFailFilterConfigDutBitSize(ffCH, 32);
    UTL_SetFailFilterConfigBitSize   (ffCH, 32);
    UTL_SetFailFilterConfigPinList   (ffCH,  0, "DQ_0_7",   UT_ON, 1);
    UTL_SetFailFilterConfigPinList   (ffCH,  1, "DQ_8_15",  UT_ON, 2);
    UTL_SetFailFilterConfigPinList   (ffCH,  2, "DQ_16_23", UT_ON, 3);
    UTL_SetFailFilterConfigPinList   (ffCH,  3, "DQ_24_31", UT_ON, 4);
      :
    UTL_SetFailListAccessDeviceChannel(flAH, 1);  /* deviceChannel=1 */
        or
    UTL_SetFailListAccessDeviceChannel(flAH, 2);  /* deviceChannel=2 */
        or
    UTL_SetFailListAccessDeviceChannel(flAH, 3);  /* deviceChannel=3 */
        or
    UTL_SetFailListAccessDeviceChannel(flAH, 4);  /* deviceChannel=4 */
