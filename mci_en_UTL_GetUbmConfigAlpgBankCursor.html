UTL_GetUbmConfigAlpgBankCursor --- Obtaining the Cursor of the ALPG Bank Number that Was Added to the Handle

SYNTAX

UbmConfigAlpgBankCursor UTL_GetUbmConfigAlpgBankCursor(UbmConfigHandle h)

ARGUMENT(S)

h : UbmConfigHandle

RETURN VALUE

UbmConfigAlpgBankCursor

DESCRIPTION

This MCI is used to obtain the cursor of the ALPG bank number that was added to the handle.

ERROR CHECK(S)

  • If an incorrect handle is specified, a runtime error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained (No change).
