Setting the Operating Conditions for Fail Filter

CAUTION

These MCIs can be used only for the following test system:

  • T5831

Fail filter does not support UTL_SetFctNoWait().

Fail filter does not support socket programs where I/O channels assigned to one DUT use multiple sites.

Fail Filter does not support socket programs using I/O branch where the DUT numbers of DUTs sharing the same pin are not specified consecutively. 

    Example: Unsupported socket program
      SEPARATE IO 2
      DUT1  =  PPS1  P1-8
      DUT2  =  PPS2  P9-16
      DUT3  =  PPS3  P49-56
      DUT4  =  PPS4  P57-64
      DUT5  =  PPS1  P1-8
      DUT6  =  PPS2  P9-16
      DUT7  =  PPS3  P49-56
      DUT8  =  PPS4  P57-64

    Example: Supported socket program
      SEPARATE IO 2
      DUT1  =  PPS1  P1-8
      DUT2  =  PPS1  P1-8
      DUT3  =  PPS2  P9-16
      DUT4  =  PPS2  P9-16
      DUT5  =  PPS3  P49-56
      DUT6  =  PPS3  P49-56
      DUT7  =  PPS4  P57-64
      DUT8  =  PPS4  P57-64
