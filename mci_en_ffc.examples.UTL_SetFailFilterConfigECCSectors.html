Example of using UTL_SetFailFilterConfigECCSectors()

If each main(core) is 1024 columns wide and each spare is 81 columns wide, the page size is 4 x (1024 + 81) = 4420.


Example 1

For the above example, set up as follows: 



    UTL_SetFailFilterConfigECCSectors(ffCH, 4, (1024 + 81), 0);
                                            ~~  ~~~~~~~~~   ~~
                                            *a     *b       *c

        *a: Number of main/spare regions
        *b: Number of columns per main region which includes spare region
        *c: No spare region


Example 2

For the above example, set up as follows: 



    UTL_SetFailFilterConfigECCSectors(ffCH, 4,    1024,     81);
                                            ~~    ~~~~      ~~
                                            *a     *b       *c

        *a: Number of main/spare regions
        *b: Number of columns per main region
        *c: Number of columns per spare region
