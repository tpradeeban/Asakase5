Example of using UTL_DeleteFailList()

  • Example 1: Delete an existing test name, and test again using the same fail filter configuration settings. 



    UTL_MeasFct(fctH);

    flAH = UTL_GetFailListAccessHandle();
    UTL_SetFailListAccessTestSelect(flAH, UT_FL_TESTSEL_NAME);
    UTL_ClearFailListAccessTestName(flAH);
    UTL_AddFailListAccessTestName  (flAH, testName);
    UTL_SetFailListAccessAllDut    (flAH, UT_ON);
    UTL_DeleteFailList             (flAH);
    UTL_DeleteHandle               (flAH);

    UTL_MeasFct(fctH);

  • Example 2: Delete an existing test name, set a different fail filter configuration, and test again. 



    UTL_MeasFct(fctH);

    flAH = UTL_GetFailListAccessHandle();
    UTL_SetFailListAccessTestSelect(flAH, UT_FL_TESTSEL_NAME);
    UTL_ClearFailListAccessTestName(flAH);
    UTL_AddFailListAccessTestName  (flAH, testName);
    UTL_SetFailListAccessAllDut    (flAH, UT_ON);
    UTL_DeleteFailList             (flAH);
    UTL_DeleteHandle               (flAH);

    /* Here, set a different fail filter configuration using
       MCIs for Setting the Operating Conditions for Fail Filter. */

    UTL_MeasFct(fctH);
