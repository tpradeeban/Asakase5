UTL_GetUbmAccessStopAddr --- Obtaining the End Address that Is Set to the Handle

SYNTAX

int UTL_GetUbmAccessStopAddr( UbmAccessHandle h, USlider *StopAddress );

ARGUMENT(S)

h : UbmAccessHandle

StopAddress : Pointer of the variable in which StopAddress is stored

RETURN VALUE

  • 1 : Read correctly.

  • 0 : No data exists in the handle.

  • -1 : Could not be read.(An error occurred.)

DESCRIPTION

This MCI is used to obtain the end address that is set to the handle.

ERROR CHECK(S)

  • If an incorrect handle is specified, a runtime error occurs.

REQUIREMENT(S)

  • UbmAccessHandle must be obtained.

RESULT(S)

  • UbmAccessHandle has been obtained (No change).
