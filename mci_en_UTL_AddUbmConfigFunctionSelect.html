UTL_AddUbmConfigFunctionSelect --- Adding the Universal Buffer Memory Function to the UbmConfig Handle

SYNTAX

void UTL_AddUbmConfigFunctionSelect( UbmConfigHandle h, RadioButton sel, RadioButton EnableCycle);

ARGUMENT(S)

h : UbmConfigHandle

sel : Universal buffer memory function Specify either of the following values: 

   For more information, refer to "Setting range for FunctionSelect".

EnableCycle : Specify the cycle in which the universal buffer memory function is enabled. 

   For more information, refer to "Setting range for Cycle".

RETURN VALUE

None

DESCRIPTION

This MCI is used to add the universal buffer memory function to the UbmConfig handle.

This MCI also sets the cycle in which the universal buffer memory function is enabled.

No universal buffer memory function is added to the UbmConfig handle immediately after the handle is obtained (after UTL_GetUbmConfigHandle() is executed) and UTL_ClearUbmConfigFunctionSelect() is executed.

If any universal buffer memory function, which cannot be set to the test system, is specified, a runtime error occurs when UTL_SendUbmConfig() is executed.

ERROR CHECK(S)

None

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained.(No change)

  • The universal buffer memory function has been added to the UbmConfig handle.

  • The UbmConfigHandle cycle has been set.
