UTL_SetUbmConfigExpDelayMode --- Setting whether to Delay the Expected Pattern

SYNTAX

void UTL_SetUbmConfigExpDelayMode( UbmConfigHandle h, CheckButton ExpDelayMode );

ARGUMENT(S)

h : UbmConfigHandle

ExpDelayMode : Specifies whether to delay the expected pattern. Specify one of the following:

  • UT_OFF : Does not cause delay.

  • UT_ON : Causes delay.

Values that can be set for ExpDelayMode
Tester nameValue that can be set
T5833UT_OFF,UT_ON
T5831UT_OFF,UT_ON
T5822UT_OFF,UT_ON
T5503HS2UT_OFF,UT_ON
T5503HSUT_OFF,UT_ON
Other than aboveUT_OFF

RETURN VALUE

None

DESCRIPTION

This MCI is used to set whether to cause delay for the number of cycles specified by UTL_SetPinExpDelay() or UTL_SetPinModeExpDelay() when capturing pass and fail information.

This MCI can be used only when UT_UBM_UNIT_SCRESULTMEMORY is specified for the UBM function.

If setting other than UT_ON is specified, UT_OFF is assumed to be specified.

The initial value when the handle is obtained is UT_OFF.

ERROR CHECK(S)

  • If an incorrect handle is specified, a runtime error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained. (No change)

  • Whether to delay the expected pattern has been set to the UbmConfig handle.

    CAUTION

    When the interleave mode is 2WAY or greater, the following restrictions apply:

    • When using the UT_DELAY_CYCLE mode of UTL_SetPinModeExpDelay(), this function does not operate correctly if a value other than the multiple of the number of interleaves is specified for the number of delay cycles.

    • The operation is not performed correctly unless all the cycles from which data is stored to UBM are enabled in the pattern program. 

      

MPAT SAMPLE

    ILMODE 2WAY

    STRAT #000

          NOP : W              XB<0
              : W

              : W R  C0

              : W R  C0
          NOP : W R  C0
              : W R  C0
          NOP : W R  C0
              : W R          <-- There is no description on the second step.
          NOP : W R  C0
              : W R  C0

END
