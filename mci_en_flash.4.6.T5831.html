T5831

  • The number of DUTs tested in parallel within a site is 4 or less.

    With a socket program specifying the I/O branch, the branched DUTs are handled as one. However, if the minimum pin for which "P" is specified is not branched, that DUT is handled as a different DUT without branches.

    When using the I/O branch, tester pins for which "P" or "PC" is specified need to satisfy all of the following conditions:

    • (a) The combination of DUTs that branch tester channels is the same.

    • (b) The tester pin numbers of branched tester channels are smaller than the tester pin numbers of un-branched tester channels.

    Also when using the mask function, the following condition needs to be satisfied in addition to all preceding conditions:

    • (c) The I/O branched DUTs branch tester channels that are assigned to all tester pins for which "P" or "PC" is specified.


  • Pins (for which "P" is specified), which are assigned to one DUT, are not assigned to multiple sites.

    If "PD" is specified for those pins, they can be assigned to multiple sites but the mask function does not operate for pins of sites other than for which "P" is specified.

CAUTION

  • A 1 site chain socket program cannot be used.
