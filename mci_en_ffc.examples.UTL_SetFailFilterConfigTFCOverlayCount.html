Example of using UTL_SetFailFilterConfigTFCOverlayCount()


Example



    UTL_SetFailFilterConfigTFCOverlayCount(ffCH, UT_ON);
    /* -> All failures (total of 35) are counted. */

    UTL_SetFailFilterConfigTFCOverlayCount(ffCH, UT_OFF);
    /* -> Failures in the same bit (in the vertical direction in the diagram) are regarded as 1 failure.
          Therefore, 14 failures are counted. */
