Editing UBM Data

To edit UBM data, perform the following procedure:

( Operation procedure)

  1. Select a cell to edit.

    To select multiple consecutive cells, all cells between the starting point and ending point are selected by performing the following operations:

    • Left-click a cell which is the starting point of the range.

    • Left-click another cell, which is the ending point of the range, while pressing the Shift key.

      When a starting point is not selected, the top-left cell is the default starting point.

  2. With cells selected, edit data from the [Edit] menu of the main window or from the pop-up menu displayed by pressing the right mouse button.

    To edit data in a single cell, the text box which is displayed by double-clicking on the cell can also be used.

    The pop-up menu is as follows:


    UBM Tool: Popup Menu

    • Multiple Cell Edit

      Edits the selected cell value.

      • Selecting the [Multiple Cell Edit] menu displays the [Multiple Cell Edit] dialog box.


        UBM Tool: [Multiple Cell Edit] dialog box

        Enter a hexadecimal number without symbol in the text box.

        To set the entered value, click the [OK] button. To cancel, click the [Cancel] button.

    • Clear

      Sets 0 (zero) for the selected cell value.

    • Copy

      Copies the selected cell value.

    • Paste

      Pastes the copied data.

      The paste operation is as follows:

      • When data is copied from a single cell, the data is pasted into the selected pasting range.

        Select the copy range.


        UBM Tool: Select the copy range.

        Select the pasting range.


        UBM Tool: Select the pasting cell.

        Paste the copied data into the selected range.


        UBM Tool: Paste the copied data into the selected range.

      • When data is copied from multiple cells, the data in the selected copy range is pasted into the selected pasting cell being used as the left top of the data range.

        Select the copy range.


        UBM Tool: Select the copy range.

        Select the pasting cell.


        UBM Tool: Select the pasting cell.

        Paste the data into the selected cell being used as the left top of the data range.


        UBM Tool: Paste the data into the selected cell being used as the left top of the data range.

Just like cells, DUT numbers, channel numbers, or addresses can be selected and edited.

  • Selecting a DUT number or channel number causes all cells of the relevant column to be selected.

    However, DUT numbers, which are split in the number of channels and displayed, cannot be selected.

  • Selecting an address causes all cells of the relevant row to be selected.

  • Just like selecting multiple consecutive cells, two or more continuous DUT numbers, channel numbers, or addresses can be selected.


    UBM Tool: Selecting multiple consecutive cells

To write edited UBM data in UBM, click the [Write] button in the main window.

The write operation for the I/O branch setting is as follows:

  • When editing only one of I/O branching DUTs, that edited data is written to all the I/O branching DUTs.

  • When editing two or more of I/O branching DUTs, data of the highest-numbered DUT of the edited DUTs is written to all the I/O branching DUTs.

  • Data is written, read, and displayed again.

CAUTION

When UBM data is not displayed, the [Write] button is grayed out to disable selection operation.
