UTL_GetUbmAccessHandle --- Obtaining the Handle

SYNTAX

UbmAccessHandle UTL_GetUbmAccessHandle( void );

ARGUMENT(S)

None

RETURN VALUE

UbmAccessHandle

DESCRIPTION

This MCI is used to obtain the handle to set the conditions that are used when accessing the universal buffer memory.

ERROR CHECK(S)

  • If the handle cannot be obtained, a system error occurs.

REQUIREMENT(S)

None

RESULT(S)

  • UbmAccessHandle has been obtained.
