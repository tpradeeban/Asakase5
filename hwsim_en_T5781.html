T5782/T5781

  1. TESIM behavior switch function

    The following function can be switched by TESIM behavior switch function.

    For more information on the procedure for switching the TESIM behavior, refer to "Details of function" in "TESIM behavior switch function."

    • Maximum test rate setting

      Set the ATFSTESIM_TG_MAXRATE environment variable.

      Available values for environment variable ATFSTESIM_TG_MAXRATE
      Environment variable valueDescription
      266MMaximum test rate is 266MHz
      166MMaximum test rate is 166MHz
      Not specifiedMaximum test rate is 166MHz

    • PPS channel number setting

      Set environment variable 'ATFSTESIM_DPU_MAXPPSCHANNEL'.

      Available values for environment variable ATFSTESIM_DPU_MAXPPSCHANNEL
      Environment variable valueDescription
      4The channel of PPS is limited to 4 for each Test Site
      8The channel of PPS is limited to 8 for each Test Site
      Not specifiedThe channel of PPS is limited to 4 for each Test Site

  2. The function limitation

    • The mode setting to the following memories is allowed but the memory function does not operate.

      • Address fail memory(AFM)

      • Bad block memory(BBM)

      • Pattern memory(PM)

      • Data Buffer Memory(DBM)

      • Universal Buffer Memory(UBM)

    • The following data is not captured in the data fail memory (DFM).

      • Fail data
