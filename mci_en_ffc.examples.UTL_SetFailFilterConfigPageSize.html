Example of using UTL_SetFailFilterConfigPageSize()

Below is a simple pattern program using UBM mode. 



    START   #400
            NOP         TS1 XB<XH YB<YH         TP<TPH
            NOP         TS1 X<XB  Y<YB  C20     ; Start iBM->BM transfer
            JBF .       TS1                     ; Wait with "JBF ." until transfer completes

    LB0:    NOP         TS1

            NOP         TS1 TRAH<TRASELD        ; Obtain address
            NOP         TS1 TRA<TRAH

            NOP         TS1 X<XB Y<YB   C31     ; Jump to the start address of UBM capture
            JNI1 .      TS1 R BMINC DS  C30     ; Capture fail information of 1 page into UBM

            NOP         TS1     C22     C20     ; Start UBM transfer and load new page of BM

            STI5 38     TS1                     ; 32 dummy cycles plus additional 40 nS
            JNI5 .      TS1                     ; Dummy looping done

            JBF .       TS1                     ; Wait with "JBF ." until transfer completes

            JNC2 FIN                            ; Jump to FIN if it is the last page
            JNI2 LB0    TS1 XB<XB+1 YB<YH       ; Next page
            JNC1 FIN                            ; Jump to FIN if it is the last block
            JNI3 LB0                            ; Next block

    FIN:    STPS                                ; Stop ALPG
