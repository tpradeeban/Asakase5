T5830

CAUTION

To use the BBM function together with the flash memory parallel measurement function, the reference function block assigned to each DUT must be the same for the flash memory parallel measurement function and the BBM function. If a DUT with different reference function block is included, a runtime error occurs.

When switching the reference function block after using the BBM function, temporarily specify the following for the function being used.

  • Flash memory parallel measurement function

    • MCI : Specify UT_FLASH_DISABLE for UTL_SetFlashModeAllDut(), and then execute UTL_SendFlash().

    • ATL : Execute the SELECT FLASH DISABLE statement.

  • BBM function

    • Specify all of the following conditions, and then execute UTL_SendFailSelPin().

      • Specify the fail capture target for UTL_SetFailSelPinTarget().

      • Specify UT_OFF for UTL_SetFailSelPin() for all tester pins.

CHTYPE42 and CHTYPE47 differ in the method for specifying tester channels in the socket program. However, the conditions where the flash memory parallel measurement function can be used are the same.

Tester channels assigned to each DUT must be specified from one of the function blocks.

The function block to which each DUT is assigned is defined as the reference function block.

Function blockTester channels of CHTYPE42Tester channels of CHTYPE47
11-24A, 33-56A1-24, 33-56

  • Socket program for CHTYPE42

    • Tester channels for each DUT are specified in child units, so they are specified from a single function block (not multiple function blocks).

      The function block to which each DUT is assigned is the reference function block.

  • Socket program for CHTYPE47

    • If tester channels are specified from a single function block (not multiple function blocks) for all DUTs, the function block to which each DUT is assigned is the reference function block.

    • If tester channels are specified from multiple function blocks for some of DUTs, the reference function block is determined by the following conditions.

      Conditions (a) to (c) have precedence in this order.

      • Condition (a): The function block containing the tester channels belonging to the tester pin to which the MATCH connection condition is specified is determined as the reference function block.

        CAUTION

        When the MATCH connection condition is specified for multiple tester pins, the tester pins must be in a single function block.

      • Condition (b): The function block containing the tester channels belonging to the tester pin specified with UTL_SetFailSelPin() is determined as the reference function block.

        CAUTION

        In the flash memory parallel measurement function, the settings with UTL_SetFailSelPin() are only used in order to determine the reference function block.

      • Condition (c): The function block containing the tester channels belonging to the lowest-numbered tester pin specified with P or PC is determined as the reference function block.

When using the flash memory parallel measurement function, the following conditions are applied to each CHTYPE setting in the socket program:

  • Conditions common to CHTYPE42 and CHTYPE47

    • Pin conditions must be specified in advance when using the flash memory parallel measurement function.

    • The number of parallel measurement DUTs must be up to 12 for all function blocks.

    • There is not a DUT branching to tester pins that use the flash memory parallel measurement function.

    CAUTION

    • The prohibition of a logical comparison is not applied to the tester pin assigned to a branching DUT.

    • The value of the hold register for the tester pin assigned to a branching DUT is 0.

  • Additional conditions for CHTYPE47

    • A DUT using a function block other than the reference function block should not be assigned to tester pins that use the flash memory parallel measurement function.

    • When the BBM is used at the same time, the reference function block for the BBM and that for the flash memory parallel measurement function must be the same.

    CAUTION

    • The prohibition of a logical comparison is not applied to tester pins assigned to a DUT that uses a function block other than the reference function block.

    • Data cannot be read when specifying tester pins assigned to a DUT that uses a function block other than the reference function block.

If any of the previously mentioned conditions are not satisfied, a runtime error can occur in the following functions:

  • MCI : UTL_SendFlash(), UTL_ReadFlashHold()

  • ATL : SELECT FLASH MODE statement, READ FLASH HOLD statement
