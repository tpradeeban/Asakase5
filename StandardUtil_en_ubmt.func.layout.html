Setting Layout of UBM Data Display Area

To set layout of the UBM data display area, perform the following procedure:

( Operation procedure)

  1. Select [Layout Setting...] from the [View] menu of the main window to display the [Layout Setting...] dialog box.


    UBM Tool: Layout Setting

  2. Specify the following data in the [Layout Setting] dialog box:

    • [Split Size]

      Set the split size when splitting and displaying UBM data.

      When the following number of channels or DUT split is set, data is split and displayed for each size specified here.

    • [Channel Count]

      Set the number of channels when splitting and displaying UBM data for each channel.

      The data is split in the number of channels specified here and displayed.

      Available values are 1, 2, and 4.

    • [Start Address]

      Set the start address to read UBM data.

      Enter a hexadecimal number without symbol.

    • [Stop Address]

      Set the end address to read UBM data.

      Enter a hexadecimal number without symbol.

  3. To set the entered value, click the [OK] button. To cancel, click the [Cancel] button.

CAUTION

  • A layout cannot be set when FIFO_2BANK is specified for the capture mode.

  • If [Start Address] or [Stop Address] is changed, the displayed UBM data is cleared.
