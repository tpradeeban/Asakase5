UTL_ClearUbmConfigJumpAddr --- Clearing the Universal Buffer Memory Jump Address and Cycle

SYNTAX

void UTL_ClearUbmConfigJumpAddr( UbmConfigHandle h, RadioButton sel, RadioButton IndexNo);

ARGUMENT(S)

h : UbmConfigHandle

sel : Universal buffer memory function Specify either of the following values: 

   For more information, refer to "Setting range for FunctionSelect".

IndexNo : Jump address index number 

   For more information, refer to "Setting range for index No".

RETURN VALUE

None

DESCRIPTION

This MCI is used to clear the jump address and cycle which were added to the UbmConfig handle by using the following MCIs:

  • UTL_SetUbmConfigJumpAddr()

  • UTL_SetUbmConfigJumpCycle()

ERROR CHECK(S)

  • If the specified handle is incorrect, a runtime error occurs.

  • If the specified universal buffer memory function is incorrect, a runtime error occurs.

  • If the specified index number is incorrect, a runtime error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained.(No change)

  • No jump address and cycle have been set in the IndexNo argument.
