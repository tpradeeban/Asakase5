UTL_SetUbmConfigPatternInvEnable --- Switching of Pattern Inverter

SYNTAX

void UTL_SetUbmConfigPatternInvEnable( UbmConfigHandle h, CheckButton onoff );

ARGUMENT(S)

h : UbmConfigHandle

onoff : Inhibits or enables pattern outputs. Select from one of the following:

  • UT_OFF -- Enables pattern outputs

  • UT_ON -- Inhibits pattern outputs.

Tester nameValues that can be set
T5833UT_OFF,UT_ON
T5831UT_OFF,UT_ON
T5830UT_OFF,UT_ON
T5822UT_OFF,UT_ON
T5781UT_OFF,UT_ON
T5761UT_OFF,UT_ON
T5588UT_OFF
T5503HS2UT_OFF,UT_ON
T5503HSUT_OFF,UT_ON
T5385UT_OFF,UT_ON
T5383UT_OFF,UT_ON
W5320UT_OFF

RETURN VALUE

None

DESCRIPTION

This is a switch of pattern inverter for UBM.

When a value other than UT_ON or UT_OFF is specified here, This MCI considers UT_ON to have been selected.

The initial value set is UT_OFF.

ERROR CHECK(S)

  • If the specified handle is incorrect, a runtime error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained.(No change)

  • The PatternInvEnable have been set to the UbmConfig handle.
