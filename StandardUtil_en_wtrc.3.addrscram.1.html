Enabling or Disabling Address Descramble Conditions

Whether to enable or disable the address descramble can be set with the address descramble enable button and address descramble area.


Wave Tracer: Address Descramble Area

Each unit of the scrambler, prescrambler, burst scrambler, block repeat scrambler, and bank address can be set to enabled or disabled.

Depending on the test system configuration, only those units for which the setting can be specified are displayed.

(Operation Procedure)

  1. Select [Address SCRAM...] from the [Condition] menu in the main window.

    The address scram dialog box is then displayed.

  2. Set the address descramble enable button to ON.

  3. Select a setting for each address descramble unit from [Default], [Enable], and [Disable].

  4. Click the [OK] button in the address descramble dialog box.

  5. Click the [Execute] button in the main window.

    Trace is executed according to the set address scrambler, and simulation waveforms are displayed.
