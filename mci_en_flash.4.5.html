Limitations on Flash Memory Parallel Measurement Function

CAUTION

For the following test systems, the flash memory parallel measurement function cannot be used in the FLG, JMZ, or JFFI instruction cycle in a pattern program.

  • T5833

  • T5831

  • T5822

For the following test systems, the flash memory parallel measurement function cannot be used with the universal buffer memory. Specify UTL_ClearUbmConfigFunctionSelect() before using the flash memory parallel measurement function.

  • T5761

  • T5588

  • T5383

For the following test systems, the flash memory parallel measurement function cannot be used with the bad block memory. Specify UT_BBM_DISABLE by using UTL_SetBbmConfigMode() before using the flash memory parallel measurement function.

  • T5761

  • T5588

The following limitations apply when using the flash memory parallel measurement function:

  1. RATE limitation

    • If the hold register load cycle is a cycle based on the flag sense instruction, the RATE limitations of the flag sense instruction must be satisfied.

    • T5781 When the IO share count is 2 or more, limitations apply to RATE in the hold register load cycle.

      RATE>=STRB setting value + 60 ns

    • T5771 Limitations apply to RATE in the following operations:

      • RATE in the hold register load cycle

      • RATE in the hold register clear cycle

      • RATE in the flash counter increment cycle

      • RATE in the flash counter clear cycle

      RATE>=STRB2 setting value + 400 ns

  2. The number of dummy cycles applied until results for which logical comparison was performed are loaded to the hold register

  3. The number of dummy cycles applied after those results are loaded to the hold register or the hold register is cleared until the forced MATCH is cancelled, the logical comparison is inhibited, that inhibition is cancelled, the driver output is inhibited, or that inhibition is cancelled.

  4. The number of dummy cycles applied after FlashCounter reaches the limit value until the forced MATCH, the inhibition of the logical comparison, or the inhibition of the driver output is enabled

  5. The number of dummy cycles applied after Bbc reaches the limit value until the forced MATCH, the inhibition of the logical comparison, or the inhibition of the driver output is enabled
