UTL_GetUbmConfigTransferSize --- Obtaining the Number of Data Items to Be Transferred to FM, which Has Been Set to the Handle

SYNTAX

int UTL_GetUbmConfigTransferSize(UbmConfigHandle h, RadioButton *TransferSize);

ARGUMENT(S)

h : UbmConfigHandle

TransferSize : Pointer of the variable in which the number of data items to be transferred to FM is stored

RETURN VALUE

  • 1 : The data has been read successfully.

  • 0 : The data does not exist in the handle.

  • -1 : Failed to read the data. (An error occurred.)

DESCRIPTION

This MCI is used to obtain the number of data items to be transferred to FM, which has been set to the handle.

ERROR CHECK(S)

  • If an incorrect handle is specified, a runtime error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained. (No change)

  • The number of data items to be transferred to FM, which was obtained from UbmConfigHandle, has been stored in the argument.
