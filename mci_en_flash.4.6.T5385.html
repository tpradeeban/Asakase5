T5385

The bit size is 1, 2, 4, 5, 8, 9, 10, 12, or 20

  • Pins to be captured

    Pins to be captured have the following restrictions:

    BITTester pin that can be captured in flash counterRestriction
    1P33Can be used for the socket with up to 4 DUTs per pin block
    2P33-34Can be used for the socket with up to 4 DUTs per pin block
    4P33-36Can be used for the socket with up to 4 DUTs per pin block
    5P33-36,P49
    8P33-40
    9 (case where P49A is used)P33-40,P49
    9 (case where P33E is used)P33-40P49 is not a flash counter count target.
    9 (case where P105 is used)-Cannot be used.
    10P33-40 ,P49,P51
    12P33-40P49, P51, P53, and P55 are not flash counter count targets
    20P33-52
    60-Cannot be used.

    • Pin block: Block that exists for each CHILD pin, in the following units: (For the T5385, 24 pin blocks in total)

      • Block of P1-52 pins

      • Block of P65-116 pins

      • Block of P129-180 pins

    • With a socket program specifying the I/O branch, the branched DUTs are handled as one.

      When using the I/O branch, tester pins for which "P" or "PC" is specified need to satisfy all of the following conditions:

      • (a) The combination of DUTs that branch tester channels is the same.

      • (b) The tester pin numbers of branched tester channels are smaller than the tester pin numbers of un-branched tester channels.

      Also when using the mask function, the following condition needs to be satisfied in addition to all preceding conditions:

      • (c) The I/O branched DUTs branch tester channels that are assigned to all tester pins for which "P" or "PC" is specified.


  • Mask target pins

    For more information on the mask target pins, refer to UTL_SetPinPatternInh().

The bit size is 1, 2, 4, 5, 8, 9, 10, or 12

  • The order of DUT numbers for pin group (*a) of tester channels must be the same for all pin blocks (*b).

    • (*a) Pin group of tester channels: Any one of P33-(33 + n - 1), P(33 + n)-(33 + nx2 - 1), ...

    • (*b) Pin block: Any one of P33-P52, P97-116, P161-180

    • The following is an example of when the DUT bit configuration is 4: 

      

Pin block      |     1     |     2     |     3     |     4     |     5    <-- Pin group number
---------------+-----------+-----------+-----------+-----------+-----------
P 33A block    | P 33- 36A | P 37- 40A | P 41- 44A | P 45- 48A | P 49- 52A
P 97A block    | P 97-100A | P101-104A | P105-108A | P109-112A | P113-116A
P161A block    | P161-164A | P165-168A | P169-172A | P173-176A | P177-180A
P 33B block    | P 33- 36B | P 37- 40B | P 41- 44B | P 45- 48B | P 49- 52B
:              | :         | :         | :         | :         | :
        

      

[Socket program that does not violate the restriction]

;;                 Pin group number    Order of DUT numbers
DUT4 = P33-36A     ;;      1                 4  --+
DUT3 = P37-40A     ;;      2                 3    |
DUT2 = P41-44A     ;;      3                 2    | <--+
DUT1 = P45-48A     ;;      4                 1  --+    | The order of DUT numbers for pin group numbers
;;                                                     | is the same for all pin blocks.
DUT8 = P33-36B     ;;      1                 4  --+    |
DUT7 = P37-40B     ;;      2                 3    | <--+
DUT6 = P41-44B     ;;      3                 2    |
DUT5 = P45-48B     ;;      4                 1  --+
        

      

[Socket program that violates the restriction]

;;                 Pin group number    Order of DUT numbers
DUT1 = P33-36A     ;;      1                 1  --+
DUT2 = P37-40A     ;;      2                 2    |
DUT3 = P41-44A     ;;      3                 3    | <--+
DUT4 = P45-48A     ;;      4                 4  --+    | The order of DUT numbers for pin group numbers
;;                                                     | is different for each pin block.
DUT8 = P33-36B     ;;      1                 4  --+    |
DUT7 = P37-40B     ;;      2                 3    | <--+
DUT6 = P41-44B     ;;      3                 2    |
DUT5 = P45-48B     ;;      4                 1  --+

    • When the order of DUT numbers for pin group numbers must be different for each pin block, for reasons such as communication restrictions between the prober, use the SYSTEMDUT statement. 

      

SYSTEMDUT1 = SITE1, DUT1
SYSTEMDUT2 = SITE1, DUT2
SYSTEMDUT3 = SITE1, DUT3
SYSTEMDUT4 = SITE1, DUT4
SYSTEMDUT5 = SITE1, DUT8
SYSTEMDUT6 = SITE1, DUT7
SYSTEMDUT7 = SITE1, DUT6
SYSTEMDUT8 = SITE1, DUT5
