CAUTION

This function can be used for the following test system:

  • W5320

This function is used to perform a DC test by using a source voltage and comparison voltage.

A unit which performs a DC test by using a source voltage and comparison voltage is called a small scale DC(SDC) unit.

The following shows how comparison and judgment are made:

  • If the voltage from the device is higher than the comparison voltage, the comparison result is 1.

  • If the voltage from the device is lower than the comparison voltage, the comparison result is 0.
