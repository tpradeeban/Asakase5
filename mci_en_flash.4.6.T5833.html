T5833

CAUTION

  • Two functions to prohibit logical comparison are provided. (Mask function)

    • Function that prohibits logical comparison regardless of the limit value of the flash counter

    • Function that prohibits logical comparison when the flash counter reaches the limit value

    The function that prohibits logical comparison regardless of the limit value of the flash counter cannot be used. If used, a runtime error occurs.

  • To use the mask function of the flash memory parallel measurement function together with that of the BBM function, the following reference function blocks assigned to each DUT must be the same:

    • Reference function block determined by the flash memory parallel measurement function

    • Reference function block determined by the BBM function

  • Pin conditions must be specified in advance when using the flash memory parallel measurement function.

If all of the following conditions (a) to (c) are satisfied, the flash memory parallel measurement function can be used without restrictions.

  • Condition (a): Each DUT uses tester channels that are assigned to a single function block.

  • Condition (b): I/O branches are not used.

  • Condition (c): The number of parallel measurement DUTs per function block is four or less.

Four function blocks that are assigned to units of 128 tester channels are provided for each child as follows:

    • Because all of conditions (a) to (c) are satisfied, the flash memory parallel measurement function can be used without restrictions.

If conditions (a) and (b) are not satisfied, the flash memory parallel measurement function can be used under the following restrictions:

  • When condition (a) is not satisfied:

    In the flash memory parallel measurement function, each DUT cannot operate by using multiple function blocks.

    Therefore, a DUT that uses multiple function blocks is assigned to one of those function blocks.

    The function block that each DUT is assigned is defined as the reference function block.

    Tester channels that use a function block other than the reference function block cannot use the flash memory parallel measurement function.

    Reference function block definition method:

    • If MATCH connection conditions have been specified for tester pins, the function block that contains tester channels belonging under the lowest numbered tester pin from those pins is the reference block.

    • If there is not a tester pin for which MATCH connection conditions have been specified, the function block that contains tester channels belonging under the lowest numbered tester pin specified with P or PC is the reference block.

    Restrictions on tester channels in a function block other than the reference function block:

    • If tester channels assigned to all tester pins specified with P or PC are not in a single function block, the following function does not operate:

      • Function that prohibits logical comparison when the flash counter reaches the limit value

      Note, however, that logical comparison is prohibited with respect only to pins in the reference function block when this function is used together with the mask function of the BBM function.

    • For the following functions, tester pins including a DUT that uses a function block other than the reference function block cannot be specified. If specified, a runtime error occurs.

      • Driver output prohibition function of the flash memory parallel measurement function

      • Data reading function of the flash memory parallel measurement function

    • If MATCH connection conditions are specified for multiple tester pins, a single function block needs to be used. If multiple function blocks are used, a runtime error occurs.

    Example socket program that does not satisfy condition (a):


    • (Description)

      • Because the MATCH connection condition is specified for the P51 tester pin, the function block that contains tester channels belonging to the P51 tester pin is determined as the reference function block for each DUT.

      • Because tester channels for DUT7 and DUT8 are selected from multiple function blocks, the following restrictions apply:

        • The flash memory parallel measurement function cannot be used for P361A and P362A of DUT7, and P233A and P234A of DUT8.

        • The function that prohibits logical comparison when the flash counter reaches the limit value does not operate.

        • For P33, P34, P39, and P40 tester pins, the data reading function of the flash memory parallel measurement function is disabled.

      • For the PD5 tester pin, the driver output prohibition function is enabled because all DUTs are selected from the reference function block.

  • When condition (b) is not satisfied:

    CAUTION

    • When specifying a socket program that uses I/O branch settings, use "MCIs that set the number of times tester hardware resources are used and signals for switching those resources".

    • When "MCIs that set the number of times tester hardware resources are used and signals for switching those resources" are used, the function that prohibits logical comparison when the flash counter reaches the limit value is disabled. (Mask function)

    If a tester pin that is specified with P or PC branches (I/O branch) to DUTs, those DUTs are handled as a single DUT.

    Method of I/O branch recognition:

    • Whether the tester pin with the MATCH connection condition setting is a branch pin

    • Whether the lowest numbered pin specified with P or PC is a branch pin, if a tester pin with the MATCH connection condition setting does not exist.

    For the following socket program, I/O branch recognition criteria differs according to the pin condition settings:


    • (Description)

      • Whether I/O branch pins are used can be recognized as follows:

        • If a MATCH connection condition is specified for the P51 tester pin, whether branch pins are used can be recognized by the P51 tester pin. Because DUT1 to DUT4 do not use branch pins, the number of parallel measurement DUTs per function block is four.

        • If a MATCH connection condition is not specified, whether branch pins are used can be recognized by the P33 tester pin. Because DUT1 and DUT2, and DUT3 and DUT4 respectively use branch pins, the number of parallel measurement DUTs per function block is two.

CAUTION

  • The driver output prohibition function cannot be specified for level driver channels.
