UTL_AddUbmConfigStrbChannel --- Setting Strobes

SYNTAX

void UTL_AddUbmConfigStrbChannel( UbmConfigHandle h, RadioButton alpgbank, RadioButton strbch );

ARGUMENT(S)

h : UbmConfigHandle

alpgbank : Bank number

Setting range for ALPG BANK No.
Tester nameRange
T58330
T58310
T5830-
T58220
T57810
T57610
T55880 or 1
T5503HS20
T5503HS0
T53850
T53830
W53200

strbch : Strobe channel

Setting range for STRB Channel No.
Range
0 or 1

RETURN VALUE

None

DESCRIPTION

This MCI is used to add the strobe channel and bank number which are used when PASS/FAIL data is captured.

Please select the STRB channel and the Bank number specified by the following MCI.

  • UTL_AddPinStrbCpeNumber()

  • UTL_AddPinWstrbCpeNumber()

CAUTION

The result of the window comparison timing (WSTRB) is stored in the strobe channel 1 side.

ERROR CHECK(S)

  • If the specified handle is incorrect, a runtime error occurs.

  • If the specified bank number is incorrect, a runtime error occurs.

  • If the specified strobe channel is incorrect, a runtime error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained.(No change)

  • Strobe channel and bank have been set to the UbmConfig handle.
