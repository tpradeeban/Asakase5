UTL_NextUbmConfigJumpIndexNo --- Obtaining the Next Jump Address Index Number from the Cursor

SYNTAX

RadioButton UTL_NextUbmConfigJumpIndexNo(UbmConfigJumpIndexNoCursor c)

ARGUMENT(S)

c : UbmConfigJumpIndexNoCursor

RETURN VALUE

Next ALPG bank number

DESCRIPTION

This MCI is used to obtain the next jump address index number from the cursor.

ERROR CHECK(S)

  • If an incorrect handle is specified, a runtime error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained (No change).
