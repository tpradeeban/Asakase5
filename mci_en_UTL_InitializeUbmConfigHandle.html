UTL_InitializeUbmConfigHandle --- Initializing the Handle

SYNTAX

void UTL_InitializeUbmConfigHandle(UbmConfigHandle h);

ARGUMENT(S)

h : UbmConfigHandle

RETURN VALUE

None

DESCRIPTION

This MCI is used to initialize the handle.

ERROR CHECK(S)

None

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained.(No change)

  • The UbmConfigHandle setting status is equal to the initial status.
