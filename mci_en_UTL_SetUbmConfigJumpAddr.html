UTL_SetUbmConfigJumpAddr --- Setting a Jump Address to the Universal Buffer Memory

SYNTAX

void UTL_SetUbmConfigJumpAddr( UbmConfigHandle h, RadioButton sel, RadioButton IndexNo,USlider JumpAddress );

ARGUMENT(S)

h : UbmConfigHandle

sel : Universal buffer memory function Specify either of the following values: 

   For more information, refer to "Setting range for FunctionSelect".

IndexNo : Jump address index number 

   For more information, refer to "Setting range for index No".

JumpAddress : Jump address 

   For more information, refer to "Setting range for JumpAddress".

RETURN VALUE

None

DESCRIPTION

This MCI is used to set a jump address for each index number to the handle.

If the index number is 0, 0 is the access start address.

ERROR CHECK(S)

  • If the specified handle is incorrect, a runtime error occurs.

  • If the specified universal buffer memory function is incorrect, a runtime error occurs.

  • If the specified index number is incorrect, a runtime error occurs.

  • If the set jump address is incorrect, a run time error occurs.

REQUIREMENT(S)

  • UbmConfigHandle must be obtained.

RESULT(S)

  • UbmConfigHandle has been obtained.(No change)

  • A jump address has been set to UbmConfigHandle.

    CAUTION
     If UT_UBM_FIFO or UT_UBM_FIFO_2BANK is specified by using UTL_SetUbmConfigCaptureMode(), the access start address for capturing pass and fail information is always 0.
