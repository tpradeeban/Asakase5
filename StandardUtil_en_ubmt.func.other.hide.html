Displaying or Hiding Each Display Area

Each display area can be hidden.

To display or hide each display area, select either option from the [View] menu in the main window.

(Operation procedure)

  • Displaying or hiding the information area

    If [Information Area] from the [View] menu of the main window is selected, the information area is displayed. If it is deselected, the information area is hidden.

  • Displaying or hiding the message display area

    If [Message Area] from the [View] menu of the main window is selected, the message display area is displayed. If it is deselected, the message display area is hidden.
